# 高效 Prompt 模板可编辑化改造（任务计划）

## 任务描述
将「AI 提效指南」页面右侧栏的静态「高效 Prompt 模板」模块替换为可编辑的 Prompt 模板库，支持新增/编辑/删除/查看/复制，并通过 localStorage 持久化。右侧栏列表需适配窄列展示（标题单行、描述两行省略），模板增多时列表区域可滚动，并提供“管理全部”入口打开 Drawer 抽屉进行长文本维护。

PRD 参考：[PRD_高效Prompt模板可编辑化.md](file:///Users/chenxunlin/工作管理/docs/PRD_高效Prompt模板可编辑化.md)

## 验收标准
- [x] AI 提效指南右侧栏展示模板列表，列表仅显示名称+描述（窄列截断省略）。
- [x] 支持新增模板（名称必填<=50字、内容必填、描述可选<=120字），保存后列表更新。
- [x] 支持编辑模板，保存后列表与详情内容更新，id/createdAt 不变，updatedAt 更新。
- [x] 支持删除模板，删除前二次确认，删除后刷新页面仍不出现。
- [x] 支持查看模板内容全文，并可一键复制到剪贴板；复制失败有明确提示与降级路径。
- [x] 数据本地持久化：刷新页面后新增/编辑/删除结果保持一致。
- [x] 模板数量较多时列表区域可滚动，不撑破页面；提供“管理全部”入口进入 Drawer 抽屉继续维护。
- [x] 不影响 AI 提效指南其他卡片与整体布局，不引入 XSS 风险（内容按纯文本渲染）。

## 问题与风险
- [ ] 风险1：localStorage 容量与清缓存导致数据丢失 - 缓解：提示“本地存储”，后续规划导入导出。
- [x] 风险2：模板内容渲染引入 XSS - 缓解：仅纯文本渲染，禁止 innerHTML。
- [x] 风险3：多标签页编辑覆盖 - 缓解：监听 storage 并提示刷新/自动刷新列表，按“后写覆盖”策略处理。
- [x] 风险4：Clipboard API 权限限制 - 缓解：失败提示 + 提供可选中文本区域便于手动复制。
